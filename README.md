# multimodal-llm-demo
# 多模态大模型入门实践项目

## 📌 项目简介
本项目旨在在个人笔记本（Lenovo Legion Y9000P，RTX 30 系显卡）上快速入门多模态大模型，完成从环境配置、模型推理到应用 Demo 封装的完整流程。  
通过使用 **Qwen-VL**, **InternVL**, 和 **VisCPM** 等开源中文多模态大模型，实现 **图文问答、OCR 识别、跨模态生成** 等典型任务。
